# image_stitching
5장의 사진중 겹치는 부분을 찾아 변형(평행이동, 회전 등등)하여 자연스럽게 stitching하는 코드입니다.

stitching 순서는 "3+4 ㅡ> 2+(3+4) ㅡ> (2+3+4)+5 ㅡ> 1+(2+3+4+5)" 입니다.
